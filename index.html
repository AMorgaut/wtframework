wtframework by AMorgaut

JavaScript frameworks detection bookmarklet

This bookmarklet detect: * YUI, dojo, JQuery, JQuery UI, Prototype, Script.aculo.us, ExtJs, Spry * Mootools, Mootools-More, Mochikit, Qooxdoo, Base2, ADSAFE, JSLint, * SWF Object (Flash), Wakanda, jmaki, AMPLE SDK, APF, 4D Ajax Framework, * Open Ajax hub & Open Ajax declared Frameworks

Install

Install it from http://wiki.github.com/AMorgaut/wtframework
